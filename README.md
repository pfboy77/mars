# Mars

A fan-made strategy board game web application inspired by Terraforming Mars, built with React and TypeScript.

## Demo

[Live Demo](https://mars-azure-beta.vercel.app/)

## Features

- Turn-based resource management
- Player resource tracking
- Terraform Rating tracking
- Undo / redo support
- Local development API server
- Responsive web UI

## Tech Stack

- React
- TypeScript
- React Router
- Create React App

## Getting Started

```bash
npm install
npm run api
npm start
```

Open http://localhost:3000 in your browser.
